#Isomo Assessment

##Project Description
This project analyzes learner engagement by comparing engagement across learning platforms (Typing and Quill) activity logs and their EFSet, Duolingo (DET), and Northstar assessment records. 

##Main objectives:
- Identify learners who are actively engaging with the platforms
- Determine whether learners with high engagement also show up in assessment records and identify possible gaps

## Repository Structure
- `data/` — cleaned datasets (raw sources are gitignored)
- `queries/` — SQL analysis, one file per question
- `analysis/` — scripts and final output
- `docs/` — design decisions, cleaning log, AI usage log
